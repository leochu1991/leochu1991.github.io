Hexo常用配置

安装完Hexo后，打开本地目录<folder>

完成后，指定文件夹目录如下：

ps : windows下使用 $ tree /f > tree.txt 生成目录树

修改配置文件_config.yml

# Site

title: Hexo #网站标题

subtitle: #网站副标题

description: #网站描述

author: John Doe #作者

language: zh-Hans #语言

timezone: Asia/Shanghai #网站时区，默认电脑时区

ps : 配置文件中 : 后面需要一个空格

# Extensions

## Plugins: https://hexo.io/plugins/

## Themes: https://hexo.io/themes/

theme: next

使用 next 主题，More info : NexT

>

# Deployment

## Docs: https://hexo.io/docs/deployment.html

deploy:

type: git

repo: https://github.com/your_github_name/your_repo_name

branch: master

message: “Site updated: YYYY-MM-DD HH:mm:ss“

# Menu configuration.

menu:

home: /

archives: /archives

categories: /categories

tags: /tags

sitemap: /sitemap.xml

commonweal: /404.html

about: /about

ps : 需要在 /source 目录下新建 categories ， tags ， about 目录和 404.html 文件

>

使用时在文件开头 Front-matter 属性设置中添加 tags: [hexo,github] 和 categories: [hexo] ，注意分类具有顺序性和层次性，也就是说 Foo, Bar 不等于 Bar, Foo；而标签没有顺序和层次。

# Avatar (put the image into next/source/images/)

# can be any image format supported by web browsers (JPEG,PNG,GIF,SVG,..)

avatar: “your_avatar_url”

ps : 可以选择 github 头像的图片地址
